Couchdb

Variable *COUCHDB-HOST* The Couchdb host to use.

Variable *COUCHDB-PORT* The Couchdb port to use.

Variable *COUCHDB-DEFAULT-DATABASE* the default database name to use.

Variable *COUCHDB-AUTH-DATABASE*

Variable *COUCHDB-SCHEME* what http scheme to use.

Variable *COUCHDB-USER* couchdb user

Variable *COUCHDB-PASSWORD* couchdb user password

HTTP API 

prev | up | next | toc

Variable *HTTP-API-ADDRESS* the listen address

Variable *HTTP-API-PORT* the port the api server listen on

Variable *HTTP-API-BASE-PATH* the base url to use for the api endpoint

Variable *HTTP-CERT-FILE* path to the http api cert providing https

Variable *HTTP-KEY-FILE* path to the http cert providing https

Variable *HTTP-SCHEME* use https or not.
